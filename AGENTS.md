# Agent Instructions

- Start every response with the wizard emoji "🧙" when working in this repo.
- Use pnpm.
- This repo is the private `me3-core` extraction scaffold. Keep it curated; do not bulk-copy production `me3-app` config.
- Never commit real secrets, `.dev.vars`, production Cloudflare IDs, production ME3 Cloud routes, or hosted subscription billing config.
- Keep Core, plugin-owned, and hosted-only boundaries explicit in code and examples.
- Run `pnpm build` after changing the web app.
